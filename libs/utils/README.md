# App Utilities

App utilities is a Node library with assorted utilities for handling UI state and processing data. Utilities specific to handling Hats and Trees should be kept in [`hats-utils`](../hats-utils/README.md)

## Utility groups

- `contract` - check ENS and whether or not address is a contract
- `events` - handling events/history of a Hat
- `form` - managing and preparing form data
- `general` - miscellaneous utilities
- `hats` - handling hats and hat data
- `ipfs` - uploading and fetching data from IPFS
- `mobile` - handling mobile-specific utilities
- `modules` - handling modules and module data
- `platform` - determining the user's browser/OS
- `subgraph` - handlers for fetching subgraph data via the SDK
- `trees` - handling trees and tree data
- `web3` - wagmi config and SDK client setup
