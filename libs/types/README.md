# Hats Types

Hats types is a library of types used across the libraries and app. Generally extending the subgraph and core SDK types provided.

## Types of types

- `authorities` - types for authorities
- `chains` - types for chain data
- `contexts` - types for context data
- `contract` - types for contract data
- `form` - types for form data
- `hat` - types for hats
- `infra` - types for infrastructure data
- `misc` - miscellaneous types
- `modules` - types for modules
- `tokens` - types for token data
- `transactions` - types for transaction data
- `trees` - types for tree data
