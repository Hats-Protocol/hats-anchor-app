# Hats Utilities

Hats Utilities is a Node library for managing and connecting to Hats and Trees.

- `authorities` - managing Authorities for hats
- `controllers` - display formatters for Hat controllers (eligibility & toggle)
- `details` - managing Hat details
- `export` - importing and exporting data from Trees
- `hats` - general Hat manipulation
- `modules` - managing Hat modules
- `tree` - managing Tree data
- `wearers` - managing Hat wearer data
